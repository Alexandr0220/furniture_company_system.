# Видео-презентация (сценарий 4–6 минут)

- Что сделано за неделю 1: БД + ERD + импорт + минимальный интерфейс/эндпоинты + отчёты

## 0:20–1:30 — База данных

## 4:30–5:30 — Как запустить
