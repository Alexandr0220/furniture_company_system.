# Проверка CRUD (быстро, через Postman)

## 1) Проверка, что сервер жив
GET http://127.0.0.1:8000/health
Ожидается: `{"status":"ok"}`

## 2) Продукция
### Получить список (просмотр)
GET http://127.0.0.1:8000/products

### Добавить (мутация)
Открыть страницу:
http://127.0.0.1:8000/products/new
Заполнить поля → Сохранить

### Редактировать (мутация)
Открыть:
http://127.0.0.1:8000/products/{id}/edit
Изменить → Сохранить
