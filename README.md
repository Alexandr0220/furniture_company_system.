# Furniture Company System — Week 1

Проект по учебной практике (1 неделя).

Цель работы — познакомиться с базовой разработкой backend-приложения, 
подключением базы данных и организацией структуры проекта.

## Используемые технологии
- Python
- FastAPI
- SQLite
- SQLAlchemy

Данный стек был выбран из-за простоты и понятности, а также удобства проверки.

## Описание проекта
Реализован простой backend для работы с данными мебельной компании.
Проект разбит на модули, каждый из которых отвечает за отдельную таблицу базы данных.
Используется REST-подход для работы с данными.

В проекте присутствуют:
- база данных
- backend-сервер
- структура модулей
- отчёты и ER-диаграмма

## Структура проекта
- app — основной код приложения
- docs — документация и ER-диаграмма
- report — отчёты по заданиям
- tests — тесты
- run.py — файл запуска сервера
- requirements.txt — зависимости проекта
