# Учебная практика — Неделя 1 (пакет сдачи)

В пакете:
- 4 отчёта (задания 1–4)
- ER-диаграмма (PDF)
- Видео: сценарий + чеклист записи (вместо самого видео)

## Как поднять проект (если вы “0”)

> Если у вас уже есть проект (код) в репозитории — следуйте шагам ниже.
> Если кода нет — добавьте папку проекта рядом с этим README.

### 1) Установить Python
Скачать Python 3.11+ и поставить галочку **Add Python to PATH**.

Проверка:
```bash
python --version
```

### 2) Скачать проект
- Открыть репозиторий → Code → Download ZIP
- Распаковать

### 3) Создать виртуальное окружение и поставить зависимости
В папке проекта:
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### 4) Создать/инициализировать БД и (опционально) импортировать Excel
```bash
python -m app.scripts.import_data
python -m app.scripts.get_sql
```

### 5) Запустить сервер
```bash
python run.py
```

Открыть:
- http://127.0.0.1:8000/products
- http://127.0.0.1:8000/workshops
